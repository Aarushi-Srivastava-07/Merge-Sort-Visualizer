Divide, Conquer, Merge

Divide: Split the array into two halves until each subarray contains a single element.

Conquer: Recursively sort each half (trivial for single elements).

Merge: Combine two sorted halves into one sorted array by repeatedly taking the smaller element.

⏱️ Time Complexity

Recurrence: T(n) = 2T(n/2) + O(n)

Solution: O(n log n) — linearithmic time.

Each level of recursion processes all n elements, and there are log₂(n) levels.
